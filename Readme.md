# Pré-requis

- Python >= 3.11
- pipenv

# Installation

Initialisation et activation d'un environnement

```
$ pipenv shell
```

Installation des dépendances

```
$ pipenv install -d
```

# Environnement

Se référer au fichier src/core/settings/env.dist

# Linting

Utiliser :

```
    $ ./lint.sh
```
